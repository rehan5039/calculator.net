Love Calculator Plugin for WordPress

Description

This love calculator evaluates the compatibility of two people based on names. The score range from 0%-100%. The higher the score the better, eg. Bill Clinton vs. Monica Lewinsky has a score of 90%. This calculator can be inserted either to the sidebar or into the post, but not both.

Compatible: WordPress 2.5 or above

License: GNU GPL license

Download (Version: 1.2)

Screenshot

Installation

  1. Unzip "love-calculator-wp-widget-1.2.zip" and upload the contained files to the "/wp-content/plugins/" directory
  2. Activate the plugin through the "Plugins" menu in WordPress

    Add this calculator to sidebar

    Install "Love Calculator" through the WordPress admin menu of Appearance or Design and then widgets.

    Add this calculator to a post or a page

    Place [calculatornet_love_calculator] in the content to insert into a post.

Faq

Can I change the look and feel of the calculator?

Yes, you can edit the code after "Edit the following to change the look and feel of this calculator" in love-calculator-wp-widget.php to change the look and feel. You need to have basic HTML understanding to change it correctly.

Can I put this calculator to both the sidebar and the post?

No, please put it to either the sidebar or the post, but not both. If put into both places, the calculator will generate javascript error.
